# Candidate Portfolio (Indore Series)

This folder packages “repo-facing” improvements (READMEs + community/CI files) so your public GitHub projects read like a single, coherent product journey.

## Recommended pinned repos

- **Indore12**: final IndiLingo (15 Indian regional languages, script-first learning).
- **Indori-Wuolingo**: original flagship for the Indori dialect.

## How the series fits together

- `Indore01` → `Indore09`: milestone snapshots (API patterns, UI experiments, exercise types, review concepts).
- `Indore10`: strong documentation/positioning draft for IndiLingo.
- `Indore11`: near-final full implementation.
- `Indore12`: final packaging, includes Netlify-friendly build script.

## What this bundle contains

- `common/`: community health files and a CI workflow you can copy into each repo
- `repos/<repo>/`: ready-to-drop files for that repo (README + common files)
